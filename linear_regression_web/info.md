# 线性回归课程 — 课件内容整理

## 一、监督学习概述

### 1.1 引入例子：房价预测
- 输入特征 x：房屋面积（平方英尺）、卧室数量等
- 输出 y：房屋价格
- 训练集：n 个训练样本 {(x^(i), y^(i)); i=1,...,n}
- 目标：学习函数 h: X→Y，使得 h(x) 能较好预测 y

### 1.2 符号约定
- x^(i)：第 i 个训练样本的输入（上标 i 是索引，非幂次）
- y^(i)：第 i 个训练样本的输出
- n：训练样本数量
- d：特征维度（不含截距项 x_0）
- X：输入空间
- Y：输出空间

### 1.3 回归 vs 分类
- 回归问题：目标变量 y 连续（如房价）
- 分类问题：目标变量 y 离散（如是/否为公寓）

## 二、线性回归模型

### 2.1 假设函数
- 单变量：h_θ(x) = θ_0 + θ_1 x
- 多变量：h_θ(x) = θ_0 + θ_1 x_1 + θ_2 x_2 + ... + θ_d x_d
- 向量形式：令 x_0 = 1，则 h(x) = θ^T x = Σ_{j=0}^{d} θ_j x_j

### 2.2 参数 θ
- θ_j 称为参数/权重
- θ_0 是截距项（偏置）
- 需要选择特征

## 三、代价函数（最小二乘法）

### 3.1 定义
$$J(θ) = \frac{1}{2} \sum_{i=1}^{n} (h_θ(x^{(i)}) - y^{(i)})^2$$

- 衡量预测值与实际值的差距
- 1/2 是为了求导方便
- 平方误差保证非负且可导

## 四、LMS算法（最小均方）= 梯度下降

### 4.1 梯度下降核心思想
- 初始化 θ
- 反复更新：θ_j := θ_j - α * ∂J(θ)/∂θ_j
- α 是学习率

### 4.2 偏导数推导
对单个样本 (x, y)：
$$\frac{\partial J(θ)}{\partial θ_j} = (h_θ(x) - y) \cdot x_j$$

### 4.3 LMS更新规则
$$θ_j := θ_j + α(y^{(i)} - h_θ(x^{(i)})) \cdot x_j^{(i)}$$

- 更新幅度与预测误差成正比
- 误差小则更新小，误差大则更新大

### 4.4 批量梯度下降（Batch GD）
$$θ_j := θ_j + α \sum_{i=1}^{n} (y^{(i)} - h_θ(x^{(i)})) x_j^{(i)}$$

- 每次迭代遍历整个训练集
- 计算量大，但收敛稳定
- J(θ) 是凸函数，必有全局最优

### 4.5 随机梯度下降（SGD）
Loop:
  for i = 1 to n:
    θ_j := θ_j + α(y^(i) - h_θ(x^(i))) x_j^(i)

- 每次只用一个样本更新
- 收敛更快（尤其大数据集）
- 在最优解附近震荡

### 4.6 学习率 α 的影响
- α 太小：收敛慢
- α 太大：可能不收敛
- J(θ) 是凸函数，梯度下降一定收敛（只要 α 不太大）

## 五、正规方程（闭式解）

### 5.1 设计矩阵 X
$$X = \begin{bmatrix} --(x^{(1)})^T-- \\ --(x^{(2)})^T-- \\ \vdots \\ --(x^{(n)})^T-- \end{bmatrix} \in \mathbb{R}^{n \times (d+1)}$$

### 5.2 目标向量
$$\vec{y} = \begin{bmatrix} y^{(1)} \\ y^{(2)} \\ \vdots \\ y^{(n)} \end{bmatrix}$$

### 5.3 矩阵形式代价函数
$$J(θ) = \frac{1}{2}(Xθ - \vec{y})^T(Xθ - \vec{y})$$

### 5.4 正规方程推导
对 J(θ) 求导并令其为零：
$$\nabla_θ J(θ) = X^T X θ - X^T \vec{y} = 0$$

解得：
$$θ = (X^T X)^{-1} X^T \vec{y}$$

### 5.5 梯度下降 vs 正规方程
| | 梯度下降 | 正规方程 |
|---|---|---|
| 需要选择 α | 是 | 否 |
| 需要迭代 | 是 | 否 |
| 计算复杂度 | O(kn d) | O(d^3) |
| 适用特征数 | 大（百万级） | 小（千级） |

### 5.6 X^T X 不可逆的情况
- 特征数多于样本数（d > n）
- 特征之间存在线性相关
- 可用正则化等方法解决

## 六、概率解释

### 6.1 模型假设
$$y^{(i)} = θ^T x^{(i)} + \epsilon^{(i)}$$

其中 ε^(i) 是误差项（IID，服从 N(0, σ²)）

### 6.2 似然函数
$$p(y^{(i)}|x^{(i)}; θ) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{(y^{(i)} - θ^T x^{(i)})^2}{2\sigma^2}\right)$$

### 6.3 最大似然估计
$$L(θ) = \prod_{i=1}^{n} p(y^{(i)}|x^{(i)}; θ)$$

取对数：
$$\ell(θ) = n \log\frac{1}{\sqrt{2\pi}\sigma} - \frac{1}{\sigma^2} \cdot \frac{1}{2}\sum_{i=1}^{n}(y^{(i)} - θ^T x^{(i)})^2$$

最大化 ℓ(θ) 等价于最小化：
$$\frac{1}{2}\sum_{i=1}^{n}(y^{(i)} - θ^T x^{(i)})^2 = J(θ)$$

### 6.4 结论
最小二乘法 = 在高斯噪声假设下的最大似然估计

## 七、欠拟合与过拟合

### 7.1 欠拟合（Underfitting）
- 模型过于简单
- 未能捕捉数据中的结构
- 高偏差（bias）

### 7.2 过拟合（Overfitting）
- 模型过于复杂
- 过度拟合训练数据中的噪声
- 在新数据上表现差
- 高方差（variance）

### 7.3 模型选择
- 特征数量需要适中
- 需要在偏差和方差之间取得平衡

## 关键公式汇总

1. **假设函数**: h_θ(x) = θ^T x
2. **代价函数**: J(θ) = (1/2)Σ(h_θ(x^(i)) - y^(i))²
3. **梯度下降**: θ_j := θ_j - α·∂J/∂θ_j
4. **LMS规则**: θ_j := θ_j + α(y^(i) - h_θ(x^(i)))x_j^(i)
5. **正规方程**: θ = (X^T X)^(-1) X^T y
6. **似然函数**: L(θ) = Π p(y^(i)|x^(i);θ)
7. **高斯分布**: p(ε) = (1/√(2π)σ) exp(-ε²/(2σ²))
