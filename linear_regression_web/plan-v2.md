# 线性回归交互式教学网页 v2 — 修复与重构计划

## 问题诊断

### 1. LaTeX 公式渲染问题（关键Bug）
在 JavaScript 字符串中，`\theta` 的 `\t` 被解释为 tab 字符而非 LaTeX 命令。
**SummarySection.tsx** 中所有公式字符串都缺少正确的转义：
- `'h(x) = \theta^T x'` ← 正确（`\\theta`）
- `'h(x) = \theta^T x'` ← 错误（`\t` 变成 tab）

同样问题可能存在于其他文件。

### 2. 单页结构 → 需要多页面

## 新页面结构（7个独立页面）

| 路由 | 页面 | 内容 |
|------|------|------|
| `/` | 课程首页 | 课程目录、章节导航、学习路径 |
| `/overview` | 监督学习概述 | 概念引入、散点图交互 |
| `/model` | 线性回归模型 | 假设函数、参数含义、拟合交互 |
| `/cost` | 代价函数 | J(θ)、等高线图、误差可视化 |
| `/gradient-descent` | 梯度下降 | LMS、批量GD vs SGD、动画演示 |
| `/normal-equation` | 正规方程 | 矩阵推导、闭式解、与GD对比 |
| `/probabilistic` | 概率解释 | 高斯假设、MLE、似然函数 |
| `/overfitting` | 模型选择 | 欠拟合/过拟合、多项式对比 |
| `/summary` | 公式速查 | 所有核心公式汇总 |
