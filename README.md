# 机器学习交互式课程

一套面向机器学习初学者的交互式学习网站。

🔗 **在线访问**：https://georgec77.github.io/machine-learning-interactive-web/

## 课程结构

网站覆盖机器学习从基础到进阶的核心主题，共分为五大部分：

- **第一部分 — 监督学习**
  - 第 1 章：线性回归
  - 第 2 章：分类与逻辑回归
  - 第 3 章：广义线性模型
  - 第 4 章：生成学习算法
  - 第 5 章：核方法
  - 第 6 章：支持向量机
- **第二部分 — 深度学习**
  - 第 7 章：深度学习
- **第三部分 — 泛化与正则化**
  - 第 8 章：泛化
  - 第 9 章：正则化与模型选择
- **第四部分 — 无监督学习**
  - 第 10 章：聚类与 K-means 算法
  - 第 11 章：EM 算法
  - 第 12 章：主成分分析
  - 第 13 章：独立成分分析
  - 第 14 章：自监督学习与基础模型
- **第五部分 — 强化学习**
  - 第 15 章：强化学习
  - 第 16 章：线性二次调节与最优控制
  - 第 17 章：策略梯度（REINFORCE）

## 内容完成状态

每个小节都有独立的网页。状态分为三档：

- ✅ **已完成**：包含完整讲解与交互式演示的内容。
- 🧪 **预览版**：主要内容已完成，但仍可能调整细节。
- 🚧 **制作中**：已有页面，但内容仍在持续完善中。

当前进度：

- 第 1 章：线性回归 —— 已完成。
- 第 2–6 章：监督学习核心主题（分类与逻辑回归、广义线性模型、生成学习算法、核方法、支持向量机）—— 预览版。
- 第 7–9 章：深度学习、泛化与正则化 —— 预览版。
- 第 10–17 章：无监督学习与强化学习 —— 制作中/持续完善。

网站会陆续更新内容，建议通过 `Ctrl+F5` 强制刷新以获取最新版本。

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- KaTeX（公式渲染）
- D3.js（交互式图表）

## 本地开发

```bash
npm install
npm run dev
```

## 部署

项目使用 GitHub Actions 自动部署到 GitHub Pages。每次 `push` 到 `main` 分支后，Actions 会自动构建并发布。
