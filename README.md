# 机器学习交互式课程

一套面向机器学习初学者的交互式学习网站。

🔗 **在线访问**：https://georgec77.github.io/machine-learning-interactive-web/

## 课程结构

网站覆盖机器学习从基础到进阶的核心主题，共分为五大部分：

- **第一部分 — 监督学习**
  - 第 1 章：线性回归
  - 第 2 章：分类与逻辑回归
  - 第 3 章：广义线性模型
  - 第 4 章：生成学习算法
  - 第 5 章：核方法
  - 第 6 章：支持向量机
- **第二部分 — 深度学习**
  - 第 7 章：深度学习
- **第三部分 — 泛化与正则化**
  - 第 8 章：泛化
  - 第 9 章：正则化与模型选择
- **第四部分 — 无监督学习**
  - 第 10 章：聚类与 K-means 算法
  - 第 11 章：EM 算法
  - 第 12 章：主成分分析
  - 第 13 章：独立成分分析
  - 第 14 章：自监督学习与基础模型
- **第五部分 — 强化学习**
  - 第 15 章：强化学习
  - 第 16 章：线性二次调节与最优控制
  - 第 17 章：策略梯度（REINFORCE）

## 内容完成状态

每个小节都有独立的网页。状态分为三档：

- ✅ **已完成**：包含完整讲解与交互式演示的内容。
- 🧪 **预览版**：主要内容已完成，但仍可能调整细节。
- 🚧 **制作中**：仅有占位页，交互式教学内容待补充。

当前已完成的章节：

- 第 1 章：线性回归（课程概览、模型表示、代价函数、梯度下降、正规方程、概率解释、过拟合）
- 第 2 章：分类与逻辑回归（课程概览、模型表示、代价函数、梯度下降、感知机、多分类、牛顿法）
- 第 3 章：广义线性模型（课程概览、指数族分布、构建 GLM、OLS/逻辑/Softmax 作为 GLM、本章总结）
- 第 4 章：生成学习算法（课程概览、生成式 vs 判别式、高斯判别分析、朴素贝叶斯）
- 第 5 章：核方法（课程概览、特征映射、特征空间 LMS、核技巧、核函数性质）
- 第 6 章：支持向量机（课程概览、间隔直观理解、SVM 理论与算法）
- 第 7 章：深度学习（课程概览、非线性模型、神经网络、现代模块、反向传播、向量化）
- 第 8 章：泛化（课程概览、偏差-方差权衡、双下降现象、样本复杂度上界）
- 第 9 章：正则化与模型选择（课程概览、正则化、隐式正则化、交叉验证、贝叶斯正则化）
- 第 10 章：聚类与 K-means 算法（课程概览、K-means 聚类）
- 第 11 章：EM 算法（课程概览、高斯混合模型的 EM、Jensen 不等式、一般 EM、GMM 再探、变分推断）
- 第 12 章：主成分分析（课程概览、PCA）
- 第 13 章：独立成分分析（课程概览、ICA）
- 第 14 章：自监督学习与基础模型（课程概览、预训练与适配、视觉预训练、大语言模型）
- 第 15 章：强化学习（课程概览、MDP、值/策略迭代、学习 MDP 模型、连续状态 MDP、算法关系）
- 第 16 章：线性二次调节与最优控制（有限时域 MDP、LQR、非线性到 LQR、LQG）
- 第 17 章：策略梯度（REINFORCE）

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- KaTeX（公式渲染）
- D3.js（交互式图表）

## 本地开发

```bash
npm install
npm run dev
```

## 部署

项目使用 GitHub Actions 自动部署到 GitHub Pages。每次 `push` 到 `main` 分支后，Actions 会自动构建并发布。
